# Unmask Setup

## Dependencies
* Unreal Engine 5.6.1
* NVIGI 1.1.0
  * Can be downloaded from https://developer.nvidia.com/rtx/in-game-inferencing#section-archive
  * Direct download: https://developer.nvidia.com/downloads/rtx/in-game-inference-sdk/nvigi_pack_ga_1_0_0.zip

## Steps
1. Download the GPT model using Download.bat at C:\Crossiont\gamedev\Unmask\Plugins\IGI\ThirdParty\nvigi_pack\plugins\sdk\data\nvigi.models\nvigi.plugin.gpt.ggml\{8E31808B-C182-4016-9ED8-64804FF5B40D}
2. Right click Unmask.unproject and generate projectfile.
3. Open  Unmask.sln with vs2022 and build the game
